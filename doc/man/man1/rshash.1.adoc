== NAME

rshash - a utility for computing various message digests

== SYNOPSIS

*{manname}* [_OPTION_...] [_FILE_...]

== DESCRIPTION

*RSHash* is a command-line utility for computing and checking various message digests.
If _FILE_ is not specified, the input is read from stdin.

== OPTIONS

=== Hash algorithms options

*-H* _NAME_, *--hash-algorithm*=_NAME_::
  Specify hash algorithm.

  _NAME_ can be:{blank}:::

    * *blake2b* and *blake2s* (BLAKE2)
    * *blake3* (BLAKE3)
    * *groestl-224*, *groestl-256*, *groestl-384* and *groestl-512* (Groestl)
    * *md2* (MD2)
    * *md4* (MD4)
    * *md5* (MD5)
    * *ripemd-160* and *ripemd-320* (RIPEMD)
    * *sha1* (SHA-1)
    * *sha224*, *sha256*, *sha384* and *sha512* (SHA-2)
    * *sha3-224*, *sha3-256*, *sha3-384* and *sha3-512* (SHA-3)
    * *streebog-256* and *streebog-512* (Streebog)
    * *whirlpool* (Whirlpool)

*--list-hash-algorithms*::
  List supported hash algorithms.

=== Verification options

*-c*, *--check*::
  Read the checksums from the file and check them.

*--quiet*::
  Don't print OK for each successfully verified file.

*--status*::
  Don't output anything, return the verification result as the exit status.

=== I/O options

*-o* _FILE_, *--output*=_FILE_::
  Output to _FILE_ instead of stdout.

*--style*=_FORMAT_::
  Specify style of the checksums.
  If this option is not specified, it is the same as selecting the default value.

  _FORMAT_ can be:{blank}:::

    * *sfv* (SFV output format, default)
    * *bsd* (BSD output format)

=== General options

*-h*, *--help*::
  Print help message and exit.

*-V*, *--version*::
  Print the version number and exit.
  When the long option is specified, additional information will also be printed.
